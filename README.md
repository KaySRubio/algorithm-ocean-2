# AlgorithmOcean2
## Basic setup instructions

To run locally:
1. `cd ui`
2. `nvm use`
3. `npm run build`
4. `npm run dev`
5. Go to http://localhost:5173/

To update:
1. `git status`
2. if needed: `git add file_name`
3. `git commit -am "commit message"`
4. `git push origin main` Or optionally create new branch for a PR
